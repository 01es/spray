###################################
high-performance network IO on Akka
###################################

*spray* is a toolkit for **REST**/**HTTP** and low-level network IO on top of |Scala|_ and |Akka|_.
Being lightweight, asynchronous, non-blocking, actor-based, modular and testable it's a great way to connect
your Akka applications with the world.
